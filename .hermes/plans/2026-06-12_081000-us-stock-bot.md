> **Goal:** Build US stock tracking bot parallel to existing IDX bot. Track institutional flow (13F), insider trading (Form 4), unusual options activity, price momentum. Telegram bot + Dashboard.

**Architecture:** Clone existing IDX project structure (FastAPI + SQLite + Telegram bot + Vue dashboard), replace data sources with US equivalents. Separate directory `analisa-saham-us/` so both run side by side.

**Tech Stack:** FastAPI, python-telegram-bot v20+, SQLite, yfinance, SEC EDGAR API (direct), finnhub free tier, Vue 3 (via CDN), Chart.js

**Models (SQLite):**

```sql
-- us_stocks — master stock list
CREATE TABLE IF NOT EXISTS us_stocks (
    symbol TEXT PRIMARY KEY,
    name TEXT,
    sector TEXT,
    industry TEXT,
    market_cap REAL,
    ipo_date TEXT,
    is_active INTEGER DEFAULT 1
);

-- daily_prices — OHLCV data
CREATE TABLE IF NOT EXISTS daily_prices (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    symbol TEXT NOT NULL,
    date TEXT NOT NULL,
    open REAL,
    high REAL,
    low REAL,
    close REAL,
    volume INTEGER,
    UNIQUE(symbol, date)
);

-- institutional_ownership — 13F filings (quarterly)
CREATE TABLE IF NOT EXISTS institutional_ownership (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    symbol TEXT NOT NULL,
    filing_date TEXT,
    period TEXT,
    total_shares REAL,
    total_value REAL,
    institution_count INTEGER,
    avg_weight REAL,
    created_at TEXT DEFAULT (datetime('now')),
    UNIQUE(symbol, period)
);

-- insider_trades — Form 4 filings
CREATE TABLE IF NOT EXISTS insider_trades (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    symbol TEXT NOT NULL,
    filing_date TEXT,
    transaction_date TEXT,
    insider_name TEXT,
    title TEXT,
    transaction_type TEXT,  -- BUY / SELL
    shares REAL,
    price REAL,
    value REAL,
    shares_held REAL,
    created_at TEXT DEFAULT (datetime('now'))
);

-- unusual_options — options flow tracking
CREATE TABLE IF NOT EXISTS unusual_options (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    symbol TEXT NOT NULL,
    trade_date TEXT,
    option_type TEXT,  -- CALL / PUT
    strike REAL,
    expiration TEXT,
    premium REAL,
    volume INTEGER,
    open_interest INTEGER,
    sentiment TEXT,  -- BULLISH / BEARISH / NEUTRAL
    source TEXT DEFAULT 'finnhub',
    created_at TEXT DEFAULT (datetime('now'))
);

-- market_movers — daily top gainers/losers/volume
CREATE TABLE IF NOT EXISTS market_movers (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    date TEXT NOT NULL,
    symbol TEXT NOT NULL,
    category TEXT,  -- gainer / loser / volume
    price REAL,
    change_pct REAL,
    volume INTEGER,
    created_at TEXT DEFAULT (datetime('now'))
);

-- watchlist
CREATE TABLE IF NOT EXISTS watchlist (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER,
    symbol TEXT NOT NULL,
    added_at TEXT DEFAULT (datetime('now')),
    UNIQUE(user_id, symbol)
);

-- users
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    telegram_id INTEGER UNIQUE,
    username TEXT,
    is_active INTEGER DEFAULT 1,
    created_at TEXT DEFAULT (datetime('now'))
);
```

## Risks & Mitigations

| Risk | Mitigation |
|------|------------|
| yfinance rate limited | Cache prices in DB, throttle requests |
| SEC EDGAR parsing changes | Fallback to Finnhub API |
| Finnhub free tier (60 req/min) | Cache aggressively, prioritize watchlist stocks |
| Options data expensive | Skip real-time, use end-of-day yfinance options |
| US market hours vs WIB | Scheduler respects US market calendar |
